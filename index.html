Página de evento

Meu projeto é um site de uma página evento chamado. Neste site, utilizei a biblioteca Sharp.js para criar um efeito na seções que exibe os principais imagem e descrição do evento. O Sharp.js é uma biblioteca JavaScript que permite redimensionar imagens de forma rápida e eficiente, o que ajudou a tornar a experiência de visualização das imagens mais suave e agradável. Além disso, também implementei uma contagem regressiva que mostra quanto tempo falta para o evento começar. Isso é feito usando JavaScript para obter a data atual e a data do evento, e depois calcular a diferença entre as duas datas para mostrar a contagem regressiva na página.

Sorteador

Meu projeto é um sorteado de números aleatórios que criei usando Grunt, Less e JavaScript. Na página inicial do site, os usuários podem clicar em um botão que gera um número aleatório. O número é então exibido na tela, logo abaixo do botão, para que o usuário possa vê-lo imediatamente. Para tornar a aparência do site mais agradável e personalizada, utilizei o Grunt e o Less. O Grunt é uma ferramenta de automação de tarefas que me permitiu automatizar processos repetitivos, como a compilação e a minificação de arquivos Less. O Less, por sua vez, é uma linguagem de estilo dinâmica que me permitiu escrever estilos para o site de forma mais eficiente.
